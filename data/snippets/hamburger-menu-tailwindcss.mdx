This is my way to create `Hamburger Menu` using `React.js` and `TailwindCSS` CSS framework. Check this out!

## Setup

1. Create a `React.js` project setup with the following syntax.

```bash:cmd
    $ npx create-react-app my-app

    # or

    $ yarn create-react-app my-app
```

2. Setup the TailwindCSS. [Read more about the steps](https://tailwindcss.com/docs/guides/create-react-app)
3. Then open the project we just created into your favorite `code editor`.
4. Next, install a package namely `react-router-dom`, here's the syntax. More about [react-router-dom](https://reactrouter.com/)

```bash:cmd
    $ npm install react-router-dom@6

    #or

    $ yarn add react-router-dom@6
```

5. Then go to the `src` folder, and create `component` folder, and then create the `Navbars.js` file.

<img src="https://naufalakbar.me/images/snippets/hamburger-menu/tailwindcss.png" alt="Snippets Assets" width="800px" align="center" style={{borderRadius: "10px"}} />

## Imports

6. Next we will start by importing everything we need.

```js:src/components/Navbars.js
import React, { useState } from "react";
import { Link } from "react-router-dom";
```

## Using useState to Open and Close Navigation

7. Now that we have content, we need a way to show it. We can use `useState` for this case. Before the return statement, add the following:

```js:src/components/Navbars.js
const [isMenuOpen, setIsMenuOpen] = useState(false);
```
