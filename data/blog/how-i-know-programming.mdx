Ini adalah pengalaman Saya tentang bagaimana Saya mengenal dunia pemrograman, bagaimana Saya masuk ke dunia pemrograman, dan bagaimana Saya menemukan jati diri Saya dalam Web Development.

## Bagaimana Saya Mengenal Dunia Pemrograman

Saya mengenal dunia pemrograman sejak Saya baru berusia 15 tahun. Dalam sistem pendidikan di Indonesia, berarti Saya masih duduk di bangku kelas 3 SMP. Saat itu Saya mengenal dunia pemrograman hanya dengan pengetahuan dasar pada awalnya, tentang membuat website sederhana hanya menggunakan HTML, CSS, dan JavaScript.

Awalnya Saya penasaran, tapi akhirnya Saya coba dan ternyata ada rasa senang saat pertama kali terjun ke dunia teknologi meski hanya basic-basic saja. Sejak saat itu Saya mulai mencari informasi tentang sekolah teknologi di Indonesia.

Setelah hampir 3 bulan mencari informasi terkait sekolah teknologi di Indonesia khususnya Kota Malang, Saya menemukan sekolah IT terbaik di Indonesia, [SMK Telkom Malang](https://smktelkom-mlg.sch.id/). Tentunya hal ini juga didukung oleh kakak-kakak Saya yang sudah lulus SMP kemudian melanjutkan pendidikannya ke SMK Telkom Malang yang membuat Saya semakin bertekad untuk melanjutkan ke SMK Telkom Malang.

## Bagaimana Saya Memasuki Dunia Pemrograman

Tak lama setelah Saya diterima dan resmi menjadi siswa SMK Telkom Malang, Saya langsung dihadapkan dengan tugas yang cukup banyak dan berat, namun semua itu bisa Saya selesaikan sampai Saya terbiasa. Di kelas 1 SMA, Saya diajari bagaimana menjadi seorang programmer mulai dari dasar dimana Saya diajari bahasa pemrograman Java.

Awalnya Saya bingung, dan mulai berpikir bahwa ini bukan passion Saya dalam mempelajari dunia pemrograman hingga hampir 1 semester Saya tidak mengerti apa yang diajarkan. Namun, di pertengahan semester 2 Saya baru menyadari bahwa Saya tidak merasa seperti Saya tidak dalam jati diri Saya, tetapi hanya tidak cocok dengan bahasa pemrograman.

## Bagaimana Saya Menemukan Jati Diri Saya dalam Web Development

Setelah Saya menyadari hal tersebut, lama kelamaan Saya mulai memahami struktur logika yang telah diajarkan dengan menggunakan bahasa pemrograman Java dari awal semester 1 hingga pertengahan semester 2. Sampai dengan buka puasa Ramadhan, Saya mulai menjelajah dunia maya. pemrograman khususnya dalam dunia Website Development.

Saya mulai belajar lagi HTML, CSS, dan JavaScript. Kemudian Saya mencoba menaikkan level yang lebih tinggi, yaitu teknologi Node.js, React.js, dan Next.js.

Jangan takut untuk belajar, carilah ilmu sebanyak-banyaknya selama mungkin dan terutama dalam dunia pemrograman itu sendiri, jangan takut untuk mencoba semua bahasa pemrograman yang ada. Tidak ada kata terlambat untuk belajar.
