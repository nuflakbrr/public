## Situs Portofolio Pribadi

Proyek pertama, Anda dapat memulai semuanya dengan membuat `Situs Web Portofolio Pribadi`. Anda bingung cara membuatnya? Jangan khawatir, Anda dapat membuatnya hanya dengan `HTML`, `CSS`, `JavaScript`. Atau Anda ingin terlihat lebih profesional, Anda dapat membuat Website Portofolio Pribadi menggunakan `React.js`, `Next.js`, `Angular.js`, `Vue.js`, dan banyak teknologi lainnya yang dapat Anda gunakan untuk buat `Situs Web Portofolio Pribadi`.

## CRUD + Otentikasi

Setelah Anda membuat `Situs Web Portofolio Pribadi`, Anda dapat langsung membuat proyek dengan menerapkan metode `CRUD` atau `Buat, Baca, Perbarui, Hapus` dan dengan penambahan fitur Otentikasi.

Berikut adalah beberapa contoh proyek yang dapat Anda buat:

- Blog
- Toko online
- Perpustakaan Online
- Rental Mobil
- dll.

## Ambil API

Kemudian Anda dapat membuat proyek menggunakan metode `Ambil API`. Ada banyak sekali API yang telah dibuat oleh developer, baik yang gratis maupun yang berbayar. Klik [link berikut](https://github.com/farizdotid/DAFTAR-API-LOKAL-INDONESIA) untuk melihat daftar API yang tersedia di Indonesia.

Berikut adalah beberapa contoh proyek yang dapat Anda buat:

- Al-Qur'an API
- Wikipedia API
- API Cuaca
- API Berita
- API Buku Resep
- API E-Commerce
- API Film
- API Musik
- dll.

## Visualisasi Algoritma

Selanjutnya, Anda dapat membuat proyek yang `memvisualisasikan` algoritme atau tantangan pengkodean yang diberikan oleh beberapa situs web yang biasanya menyediakan tantangan pengkodean. Karena mempelajari algoritme atau mengerjakan tantangan di situs web coding challenge ini sebenarnya dapat meningkatkan logika pemrograman kita.

Ini adalah beberapa contoh tantangan pengkodean situs web:

- [Codewars](https://www.codewars.com/)
- [Hackerrank](https://www.hackerrank.com/)
- [FreeCodeCamp](https://www.freecodecamp.org/)
- dll.

Jika kita mempelajari algoritma yang diberikan oleh website coding challenge, biasanya hanya berbentuk `tekstual`. Untuk membuat proyek `Visualisasi Algoritma` lebih menarik, kami mencoba membuatnya dengan `visualisasi` sehingga dapat menghasilkan `visualisasi interaktif`.

## PROYEK NYATA

Kemudian untuk proyek terakhir dan menurut Saya, Anda harus membuatnya. Karena dalam proyek ini Anda melibatkan `klien` tetapi prosesnya tidak harus dianggap `profesional`. Karena di sinilah Anda menggunakan logika pemrograman, keterampilan pengkodean, algoritma, pemecahan masalah, dll. Proyek seperti apa? Sederhana.

Anda bisa membuat website berupa landing page, katalog, company profile, dll. Bagaimana cara menemukannya? Cara termudah adalah dengan mencarinya yang ada di sekitar kita. Kami dapat membantu calon `klien` kami yang memiliki bisnis tetapi belum memiliki website.
